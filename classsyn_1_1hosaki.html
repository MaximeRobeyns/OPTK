Detailed Description

The Hosaki function has the following equation:

\[ f(\mathbf{x}) = \bigg( 1 - 8x_1 + 7x_1^2 - \frac{7}{3}x_1^3 + \frac{1}{4}x_1^4 \bigg) x_2^2 e^{-x_2}, \]

subject to \(0 \le x_1 \le 5\) and \(0 \le x_2 \le 6\) with a global minimum located at \(\mathbf{x}^* = (4,2)\) with value \(f(\mathbf{x}^*) = -2.3458\).
