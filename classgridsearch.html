◆ add_to_trials()

void gridsearch::add_to_trials ( int  param_id,
inst::node *  n 
)
private

This will add a parameter configuration allocated on the heap to the gridsearch::trials map, so that it may be freed later by gridsearch's destructor. If a trial configuration already exists with the same param_id, those values will be freed, and the value in the map will be replaced.

Parameters
param_idThe identifier for this trial.
nThe root node of the concrete parameter settings.

Expands the lists of parameters into a format from whcih it is easier to generate all possible configurations sequentially.

Parameters
